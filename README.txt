Il progetto è organizzato in vari file .py, per eseguire il run.py è strettamente necessaria la cartella "checkpoints" dove sono caricati i pesi dei vari modelli preaddestrati ed il file models.py il quale ha al suo interno la funzione che ritorna i modelli utilizzati.
Gli altri due file "data.py" e train.py" sono stati utilizzati per il training dei modelli.
